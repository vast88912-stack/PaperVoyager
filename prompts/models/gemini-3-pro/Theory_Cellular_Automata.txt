**Project:** Cellular Automata Universe (Gold Standard)
**Role:** Senior Creative Technologist.
**Objective:** Build a high-complexity, single-file React+TS teaching dashboard that explores Cellular Automata (Game of Life, Wireworld, Elementary CA).

**Global UX Spec:**
- Layout: Landing Page (Hero) -> Dashboard (Sidebar + Main + Inspector).
- Theme: Scientific Dark Mode (Pixel Art / Digital Life).
- Navigation: Ruleset Selector.
- Inspector: "Population Stats" and "Generation Speed".

**Module Plan (Landing + 5 Modules):**
1. **Module 0: Hero / Emergence**
   - Visual: A Gosper Glider Gun creating infinite life.
   - Text: "Simple rules, infinite complexity."

2. **Module 1: Elementary CA (Rule 30)**
   - Concept: 1D Automata.
   - Interactive: Set the 8-bit rule (0-255).
   - Visualization: Waterfall plot (Time goes down).
   - Observation: Chaos from order (Rule 30).

3. **Module 2: Conway's Game of Life**
   - Concept: 2D Life.
   - Interactive: Draw on grid.
   - Presets: Glider, Pulsar, Spaceship.
   - Controls: Play/Pause, Speed.

4. **Module 3: Wireworld (Logic Gates)**
   - Concept: Computing with cells.
   - States: Empty, Head, Tail, Conductor.
   - Interactive: Build a logical OR gate or a Clock.
   - Visual: Electrons flowing through wires.

5. **Module 4: Predator-Prey (Wa-Tor)**
   - Concept: Ecosystem.
   - States: Grass, Rabbit, Fox.
   - Simulation: Population cycles (Foxes eat Rabbits, Rabbits eat Grass).
   - Chart: Oscillating population graph.

6. **Module 5: The "God Mode" Challenge**
   - Task: "Create a Glider."
   - Goal: In Game of Life, paint a pattern that moves diagonally forever.
   - Scoring: Pattern longevity.

**Scoring & Gamification:**
- **Life Creator**: Earned by keeping a population alive for 1000 gens.
- **Logic Engineer**: Built a working wireworld circuit.
