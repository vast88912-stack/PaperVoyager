**Project:** Dynamic Programming Lab (Gold Standard)
**Role:** Senior Creative Technologist.
**Objective:** Build a high-complexity, single-file React+TS teaching dashboard that demystifies Dynamic Programming (DP) by visualizing the "Table Filling" and "Memoization" process.

**Global UX Spec:**
- Layout: Landing Page (Hero) -> Dashboard (Sidebar + Main + Inspector).
- Theme: Scientific Dark Mode (Blue/Indigo palette).
- Navigation: Problem Selector (Knapsack, LCS, Edit Distance).
- Inspector: "Recursion Stack" vs "DP Table" viewer.

**Module Plan (Landing + 5 Modules):**
1. **Module 0: Hero / The Time Traveler**
   - Visual: A fractal tree growing and then collapsing into a simple line (Memoization).
   - Text: "Those who cannot remember the past are condemned to repeat it. DP remembers."

2. **Module 1: The Fibonacci Spiral (Memoization)**
   - Concept: Top-Down vs Bottom-Up.
   - Visualization: A Recursion Tree that turns "Green" when a value is reused (pruned).
   - Compare: Naive Recursion (Exponential) vs Memoized (Linear). Show the explosion of nodes in Naive.

3. **Module 2: The Knapsack Problem (0/1)**
   - Concept: Optimal Substructure.
   - Visualization: A 2D Grid (Items vs Capacity).
   - Interactive: Click cells to see dependencies ("To fill this, I looked at that cell").
   - Controls: Drag items into a bag to see if human intuition matches the algorithm.

4. **Module 3: String DNA (Longest Common Subsequence)**
   - Concept: Sequence alignment.
   - Visualization: "String Grid" showing matches. Trace the "Backtracking" path to reconstruct the solution.
   - Application: Simulate a mini DNA comparison tool.

5. **Module 4: Seam Carving (Content-Aware Image Resize)**
   - Concept: Real-world DP application.
   - Visualization: A small image (pixel grid). Calculate "Energy" of pixels. Find the "Lowest Energy Path" (seam) from top to bottom.
   - Interactive: Remove seams one by one to shrink the image without distorting objects.

6. **Module 5: The DP Challenge**
   - Concept: Fill the Table.
   - Task: "Predict the value of cell [3][5] in this Knapsack table."
   - Scoring: Points for correct manual calculation.

**Scoring & Gamification:**
- **Optimization Score**: How many redundant calculations did you save using Memoization?
- **Unlockable**: "Seam Carving Mode" unlocks after finishing Knapsack.

**Implementation Notes:**
- Use `lucide-react` for icons.

- **Visualization**: The "DP Table" should be highly interactive. Hovering a cell highlights its "Parents".
- **Performance**: For Seam Carving, use a small grid (e.g., 20x20) or Canvas if larger.
