**Project:** CPU Scheduler Simulator (Gold Standard)
**Role:** Senior Creative Technologist.
**Objective:** Build a high-complexity, single-file React+TS teaching dashboard that visualizes Operating System Process Scheduling (FIFO, Round Robin, SJF).

**Global UX Spec:**
- Layout: Landing Page (Hero) -> Dashboard (Sidebar + Main + Inspector).
- Theme: Scientific Dark Mode (Terminal Green / Retro OS).
- Navigation: Algorithm Selector.
- Inspector: "Process Table" and "Gantt Chart".

**Module Plan (Landing + 5 Modules):**
1. **Module 0: Hero / The Time Slicer**
   - Visual: A CPU core rapidly switching context between tasks.
   - Text: "The illusion of multitasking."

2. **Module 1: The Process Queue**
   - Concept: Arrival Time, Burst Time.
   - Interactive: "Add Process" (give it a name and duration).
   - Visual: Processes waiting in line.

3. **Module 2: First-Come, First-Served (FIFO)**
   - Concept: No preemption.
   - Simulation: The Convoy Effect.
   - Visual: A long process blocks everyone. Average Wait Time explodes.

4. **Module 3: Shortest Job First (SJF)**
   - Concept: Optimization.
   - Simulation: Reorder the queue. Small tasks go first.
   - Metric: Drastic drop in Average Wait Time.

5. **Module 4: Round Robin (Time Quantum)**
   - Concept: Fairness.
   - Controls: Slider for Time Quantum (Q).
   - Animation: Processes get sliced.
   - Tradeoff: Small Q = High Overhead. Large Q = FIFO.

6. **Module 5: The Scheduler Challenge**
   - Task: "Keep the Latency Low."
   - Goal: Incoming stream of processes. You manually pause/resume tasks. Keep average wait time below 5s.
   - Scoring: Survival time.

**Scoring & Gamification:**
- **Kernel Developer**: Mastered Round Robin.
- **Efficiency Guru**: Minimized wait time.
